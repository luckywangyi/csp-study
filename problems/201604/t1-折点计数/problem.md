# 折点计数

**期次**：201604  
**题号**：T1  

## 题目描述

给定由 $n$ 个**互不相同**的整数组成的序列 $a_1,a_2,\ldots,a_n$。若下标 $i$（$2 \le i \le n-1$）满足：$a_i$ **同时严格大于** $a_{i-1}$ 与 $a_{i+1}$，**或同时严格小于** $a_{i-1}$ 与 $a_{i+1}$，则称 $a_i$ 为一个**折点**。请求出折点的个数。

## 输入格式

- 第一行一个正整数 $n$。
- 第二行 $n$ 个互不相同的整数，表示序列 $a_1,\ldots,a_n$。

## 输出格式

一行一个整数，表示折点个数。

## 样例

### 输入

```
5
1 5 2 6 3
```

### 输出

```
3
```

**说明**：$5$、$2$、$6$ 为折点（分别为峰、谷、峰）。

### 输入

```
3
1 2 3
```

### 输出

```
0
```

## 数据范围与提示

- $n \ge 3$ 时中间位置才可能为折点；$n<3$ 时通常为 0（以官方数据为准）。
- 元素互不相同，无需考虑等于的情况。
