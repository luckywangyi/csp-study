# 图像旋转

**期次**：201503  
**题号**：T1  

## 题目描述

给定一个 $n$ 行 $m$ 列的矩阵，将其**顺时针旋转 $90^\circ$** 后输出新矩阵（共 $m$ 行 $n$ 列）。旋转后原矩阵左上角的元素位于新矩阵的右上角一侧（见样例）。

## 输入格式

- 第一行两个正整数 $n$、$m$，用空格分隔。
- 接下来 $n$ 行，每行 $m$ 个整数，表示原矩阵（行优先）。

## 输出格式

输出 $m$ 行，每行 $n$ 个整数，相邻整数之间用一个空格分隔，行末无多余空格（或允许行末空格以评测机为准）。

## 样例

### 输入

```
2 3
1 2 3
4 5 6
```

### 输出

```
4 1
5 2
6 3
```

## 数据范围与提示

- $1 \le n, m \le 1000$ 量级常见（以官方题面为准）。
- 顺时针 $90^\circ$：新矩阵第 $i$ 行第 $j$ 列对应原矩阵第 $(n-j)$ 行第 $i$ 列（下标从 0 或从 1 推导时注意边界）。
- 亦可先转置再左右翻转，等价于顺时针 90°。
