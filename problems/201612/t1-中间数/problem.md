# 中间数

**期次**：201612  
**题号**：T1  

## 题目描述

给定 $n$ 个整数。若存在一个整数 $x$，使得序列中**严格小于** $x$ 的数的个数与**严格大于** $x$ 的数的个数**相等**，则称 $x$ 为该序列的一个**中间数**。若存在多个满足条件的 $x$，输出其中**最小**的一个；若不存在，输出 `-1`。

（注意：与 $x$ **相等**的数既不记入「比它小」也不记入「比它大」。）

## 输入格式

- 第一行一个正整数 $n$。
- 第二行 $n$ 个整数（以空格分隔）。

## 输出格式

一行一个整数：最小的中间数，或 `-1`。

## 样例

### 输入

```
3
3 1 2
```

### 输出

```
2
```

**说明**：小于 2 的数有 1 个（1），大于 2 的数有 1 个（3）。

### 输入

```
2
1 2
```

### 输出

```
-1
```

## 数据范围与提示

- $n$ 与数值范围以官方题面为准；可先排序，按**数值分组**或枚举每个不同取值检验左右个数。
- 对每个不同的 $v$，设小于 $v$ 的个数为 $L$，大于 $v$ 的个数为 $R$，当且仅当 $L=R$ 时 $v$ 合法。
