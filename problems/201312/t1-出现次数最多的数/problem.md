# 出现次数最多的数

**期次**：201312  
**题号**：T1  

## 题目描述

给定 $n$ 个正整数，请找出其中**出现次数最多**的数。若有多个数出现次数相同且均为最多，输出其中**最小**的那个。

## 输入格式

- 第一行一个正整数 $n$，表示数的个数。
- 第二行 $n$ 个正整数，相邻整数之间用一个空格分隔。

## 输出格式

输出一行一个整数，即出现次数最多的数（并列时取最小）。

## 样例

### 输入

```
6
10 1 10 20 30 20
```

### 输出

```
10
```

**说明**：10 与 20 均出现 2 次，次数最多，其中较小的是 10。

### 输入

```
4
1 2 3 4
```

### 输出

```
1
```

## 数据范围与提示

- $1 \le n \le 1000$（以常见 CSP 数据范围为准，具体以官方题面为准）。
- 给出的数均为正整数，可用 `map` 或数组统计频次。
- 注意并列时取**数值最小**，而非先出现的数。
