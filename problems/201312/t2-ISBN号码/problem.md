# ISBN号码

**期次**：201312
**题号**：T2

## 题目描述

每一本正式出版的图书都有一个 ISBN 号码与之对应。ISBN 号码的格式为 `x-xxx-xxxxx-x`，其中 `x` 为 0-9 的数字。

ISBN 号码的最后一位是校验码，校验码的计算方法如下：将前 9 位数字分别乘以 1, 2, 3, ..., 9，将各乘积之和对 11 取余，所得余数即为校验码。如果余数为 10，则校验码为 `X`。

给定一个 ISBN 号码，判断其校验码是否正确。

## 输入格式

一行，一个 ISBN 号码（格式为 `x-xxx-xxxxx-x`，最后一位为校验码，可能是 0-9 或 X）。

## 输出格式

如果校验码正确，输出 `Right`。
如果校验码不正确，输出正确的 ISBN 号码（用正确的校验码替换原来的校验码）。

## 样例

### 输入

```
0-670-82162-4
```

### 输出

```
Right
```

### 输入

```
0-670-82162-0
```

### 输出

```
0-670-82162-4
```

## 数据范围与提示

- 输入保证格式正确
- 校验码范围：0-9 或 X
